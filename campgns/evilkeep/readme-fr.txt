EVIL KEEPER (TM)

Signification des boutons
=========================

BEENDEN : Quitter
MONSTER GUIDE : Guide des créatures
WICHTIGE INFOS : Informations importantes
PATCH VERSION : Installer la version patchée (sauf au menu principal)
NORMALE VERSION : Installer la version normale (sauf au menu principal) - NE PAS TOUCHER pour le moment !!!
ZURÜCK : Retour
WEITER : Suivant
QUICK INSTALL : Pour accéder à la page d'installation rapide
ANLEITUNG (pas un bouton) : Guide
INSTALL (pas un bouton) : Installation

Notes importantes
=================

Lorsque vous cliquez sur PATCH VERSION ou NORMALE VERSION pour installer le jeu, une fenêtre apparaîtra vous indiquant
d'indiquer le chemin d'accès qui mène au dossier Keeper sur votre disque dur dans la fenêtre suivante.
Cliquez sur OK et dans la fenêtre suivante, mettez le chemin d'accès comme prévu dans la barre et cliquez sur Unzip.

Attention: le programme écrase les fichiers déjà existants. Pour retrouver les niveaux de Dungeon Keeper original,
vous devrez les reprendre depuis le CD de Dungeon Keeper dans le dossier Keeper\Levels. Pour retrouver les voix originaux,
faites de même mais dans le dossier Keeper\Sound\Atlas\French. Pour le texte, faites de même mais dans le dossier Keeper\Data.
Sinon, gardez une copie des fichiers quelque part pour les remettre comme avant par la suite.

Vous pouvez sinon indiquer un autre endroit où dézipper le tout et faire du copier-coller manuellement.

Aussi, ne prenez que la version IMPROVED.ZIP. C'est la seule version que j'ai corrigée à ce jour. Je ne pense pas que
je vais corriger STANDARD.ZIP, je crois que la plupart d'entre vous ont déjà patchés le jeu depuis longtemps ou avez
la version Gold, n'est-ce pas?
Alors, prenez la version IMPROVED.ZIP.

Bon jeu!
